# Power BI Global Sales Dashboard

This project presents a Power BI dashboard analyzing global sales performance using the AdventureWorks dataset.

## Key KPIs
- Total Sales: $26M+
- Average Margin: 41%
- 59K+ sales line items
- 26K+ sales orders

## Dashboard Pages
1. Sales Overview
2. Regional Performance
3. Product & Country Sales Insights

Power BI  
DAX
